gm-map

Parent

None

Scope Map Parameters LatLng to update the map center with. Number to update the map heading with. Number to update the map tilt with. Number to update the map zoom with. MapTypeId to update the map type with.

To be displayed, Map require a value for each of these parameters: center, mapTypeId and zoom. It can be provided by options and / or watched expressions.
