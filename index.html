Angular-google-maps-native

AngularJS directives for the Google Maps Javascript API using the native way

View on GitHub Download .zip Download .tar.gz

Angular Google Maps Native is a set of AngularJS directives which allows to create Google Maps using the Google Maps Javascript API syntax.

The directives syntax is as close as possible of the Google Maps library to not only provide a limited subset of the API.

It is based on the Google Maps Javascript API version 3.

Example

Angular-google-maps-native is maintained by jbdemonte, author of the gmap3 plugin for jQuery.
